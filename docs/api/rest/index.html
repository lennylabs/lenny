Lenny REST API Reference

Interactive API explorer powered by Swagger UI. This page loads the OpenAPI specification from the running Lenny gateway. ← Back to API Reference

How to use this page:
  1. Click Authorize (lock icon) and enter your Bearer token to authenticate API requests.
  2. Expand any endpoint to view its parameters, request body schema, and response format.
  3. Click Try it out on any endpoint to send a live request to your Lenny gateway.
  4. Use the Models section at the bottom to browse all schema definitions.

Tip: Override the spec URL with the ?url= query parameter. For example: ?url=https://lenny.example.com/openapi.yaml. Default: /openapi.yaml.
